Verbinding met dashboard

De extension verzendt geanonimiseerde metadata (geen prompttekst) naar uw eigen PromptGuard-dashboard.

Gedrag

Bepaal hoe de extension reageert wanneer gevoelige data is gedetecteerd.

Privacy

De extension scant prompts lokaal in uw browser. Alleen geanonimiseerde tellingen worden naar het dashboard verzonden — nooit de prompttekst zelf.

✓
Lokale detectie
Alle pattern-matching gebeurt in uw browser, zonder netwerkcalls.
✓
Open source
De code van deze extension is volledig leesbaar voor uw IT-team.
✓
Geen prompttekst
Naar het dashboard gaat alleen "X BSN's gedetecteerd" — nooit de inhoud.
✓
Geanonimiseerde gebruiker-ID
Een SHA-256-hash, niet uw e-mailadres of naam.

Lokale data wissen
